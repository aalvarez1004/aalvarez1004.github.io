Els projectes que farem al primer trimestre estaran relacionats amb dades que estan en servidors i que es consulten mitjançant claus API o que es troben lliures en format JSON (JavaScript object notation) i a partir de les quals farem una representació artística.

El JavaScript és el codi principal que ens permet connectar a un servidor o fer una funció específica, com per exemple agafar dades JSON o resoldre una equació de segon grau, o qualsevol cosa interactiva.
